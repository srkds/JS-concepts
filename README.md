### Concat

```bash
const ourStr = "I come first. " + "I come second.";
// I come first. I come second.
```

concat two string using `+`sign
**Concatenating Strings with the Plus Equals Operator**

```bash
let ourStr = "I come first. ";
ourStr += "I come second.";
// I come first. I come second.
```
